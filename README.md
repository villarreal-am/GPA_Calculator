# GPA_Calculator

## Overview📖
A weighted GPA calculator that runs inside a terminal window. You are prompted to enter your grades and weight for each grade. When you are finished, type DONE and the program will calcuate your weighted GPA based off of the grades and weights provided. To use this program, run the file through a terminal window. You should be prompted to start entering your grades.
